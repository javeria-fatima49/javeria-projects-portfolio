Calculator

The calculator features a user-friendly interface with large, easy-to-read buttons and a clear display area for input and results. ATM Simulator

The ATM Simulator project simulates the core functionalities of an automated teller machine, providing a realistic experience of banking transactions. Users can securely log in using a PIN and access a range of services, including checking their account balance, withdrawing cash, depositing funds, and viewing their transaction history. The system ensures the security of user data with encrypted storage and robust error handling. This project also includes a backend component for managing user accounts and transactions, demonstrating skills in both frontend and backend development.

  • Try it with npx:npx jiya26-atm
